# Клиент-серверный общий чат
Написан на сетевом фреймворке twisted.
Интерфейс на Pyqt5
## Возможности
 - Отправка сообщений всем пользователям в чате или же кому-то конкретно
 - Отправка файлов размером до 100 мб всем пользователям в чате или же кому-то отдельно
## Файлы
 - server.py сервер
 - client.py клиент
 - chat.py интерфейс чата
